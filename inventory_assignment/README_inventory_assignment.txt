GENERAL INFORMATION

Title of Dataset: webfiles-samples

Date of data collection: This data was collected from the webfiles-samples directory on 2023-10-30 to get metadata information about each of the file. 

Geographic location of data collection: Ann Arbor, MI 48104 USA

Information about funding sources that supported the collection of the data: This was collected using open source information and resources provided by the University of Michigan. 


File List: audio, image, pdf, presentation, video, web-files-small-mdtadata.csv

Information about data columns: 
relative_path: Provides the relative path to each file in the foler. The relative path is the location of a file relative to the current working directory. 
file_name: Provides the name of the file. 
file_extention: lists the what type of file the file in question is. This lets users know what kidn of files they are working with. 
file_size: Provides the size of the file in bytes.
mod_time: Lists the time that the file was last modified in YYYY-MM-DD format.
file_integrity_info: Provides and md5 checksum for the files in the inventory.
